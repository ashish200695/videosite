# Horror Story Submission Platform with Dailymotion Monetization

A Next.js-based web platform for hosting horror story video content with contributor submission, moderation, and revenue-sharing capabilities.

## Features

### Public Site
- Homepage with featured/trending horror stories
- Story detail pages with embedded Dailymotion players
- Browse by category/tags
- Search functionality
- Mobile-first responsive design

### Contributor System
- User signup/login
- Video submission form
- Submission status tracking
- Revenue dashboard
- Payout management

### Admin Dashboard
- Moderation queue
- Approve/reject submissions
- Platform analytics
- Payout management

## Tech Stack
- Frontend: Next.js 14, React 18, Tailwind CSS
- Backend: Node.js with Next.js API Routes
- Database: PostgreSQL with Prisma ORM
- Authentication: NextAuth.js
- Video Hosting: Dailymotion API

## Getting Started

npm install
npm run dev

Open http://localhost:3000
